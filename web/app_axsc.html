Em um mundo onde os seus dados valem, a privacidade se torna algo raro. O AXSC é um chat baseado em terminal com foco extremo em privacidade, com uma política sem logs, open-source e privacidade de ponta a ponta se torna uma verdadeira terra sem lei.

Criptografia de ponta-a-ponta

Todo o sistema de criptografia do AXSC é feito localmente utilizando um Hash gerado pela chave da sala na qual a sua mensagem foi enviada. Isso garante que o intermediário nunca irá conseguir descobrir o conteúdo das mensagens, permitindo que, apenas os outros usuários da sala tenham acesso a descriptografar a mensagem, devido a eles possuirem o Hash da Sala.

Criptografia AES-256-CBC


O AXSC utiliza o Advanced Encryption Standard 256Bits com Ciper Block Chaining, tal algoritmo é amplamente utilizado pelo Governo dos Estados Unidos para manter o sigilo de dados sensíveis.

Junto a isso, o CBC garante que cada transmissão de dados seja única, dificultando ainda mais uma descriptografia por reconhecimento de padrões.
